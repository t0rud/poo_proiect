# Sistem de gestiune al unei cafenele

Pentru acest proiect, am ales să implementez un sistem de gestiune a comenzilor și produselor dintr-o cafenea, deoarece este o temă care îmi permite să modelez natural conceptele predate. Astfel, pentru prima parte (tema 2) voi crea o ierarhie de produse, cum ar fi băuturi și patiserie, pentru a mă folosi de moștenire, funcții virtuale și excepții proprii, iar pentru tema 3 voi extinde codul adăugând un inventar generic cu clase template și design patterns, cum ar fi Builder pentru personalizarea cafelei și Singleton pentru managementul aplicației.
